Proxy Server
Create a web proxy server that can be connected by a single client and would only allow http requests. The proxy server should be able to cache up to five recent websites.

Put all files to one directory on cse machine.
Type "make" to run makefile.
After makefile runs, proxy server and client will be compiled.
The executable files called "pyserver" and "client".
Use ./pyserver <Port Number> and ./client <Port Number> to run them.
Please run pyserver and client with a same valid port number.
After using the program, you can use make clean to remove all executable files.
